Stratus is a browser-based Unity-like game engine and editor designed to provide hobbyist game developers with the ability to create, publish and play games all inside their browser. Games can be created easily using the built-in editor by adding entities and components to a scene. JavaScript can be used to create new components with the Three.js and Ammo.js(Bullet physics) libraries.
